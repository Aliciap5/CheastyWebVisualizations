Summary: Latitude vs. Weather

The purpose of this project was to analyze how weather changes as you get closer to the equator. This analysis uses data pulled from the OpenWeatherMap API to assemble a dataset on over 500 cities.

After assembling the dataset Matplotlib was used to plot various aspects of the weather in comparison to thier latitude. Factors analyised were temperature, cloudiness, wind speed, and humidity. This site provides the source data and visulazations created as part of the anaysis, as well as explanations and descreptions of any trends and correlations witnessed.
